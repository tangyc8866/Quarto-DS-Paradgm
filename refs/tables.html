6.2 表格示例2: 由数据框构造表格

代码

#| label: tbl-tab3-2
#| tbl-cap: '遗传连接模型例子中观测到的频数 $y_i$ 和频率 $p(y_i|\pi)$，$i=1, \dots ,4$，197个动物.'
#| echo: FALSE
#| results: "asis"
tab.pis <- data.frame(
  C = c("Frequency $y_i$", "Probability $p(y_i\\mid\\pi)$"),
  V1 = c("125", "$\\frac{1}{2}+\\frac{\\pi}{4}$"),
  V2 = c("18","$\\frac{1}{4}(1-\\pi)$"),
  V3 = c("20","$\\frac{1}{4}(1-\\pi)$"),
  V4 = c("34","$\\frac{1}{4}$")
)
names(tab.pis) <- c("Category", "1", "2", "3", "4") 

knitr::kable(tab.pis, escape = FALSE)

输出为

注释

表格中的题图通过tbl-cap:由文本引用生成.

6.5 数据子表的并置与堆叠

在R的代码块选项中设置tbl-subcap: true或tbl-subcap加上子表的图例，并添加layout-ncol: 2就可获得二个子表的并置.

#| label: tbl-tables
#| tbl-cap: "两个表格并置示例2"
#| tbl-subcap:
#|   - "Cars数据"
#|   - "Pressure数据"
#| layout-ncol: 2

library(knitr)
kable(head(cars))
kable(head(pressure))

## 数据子表的并置与堆叠 {#sec-tables-5}

在`R`的代码块选项中设置`tbl-subcap: true`或`tbl-subcap`加上子表的图例，并添加`layout-ncol: 2`就可获得二个子表的并置.

```r
#| label: tbl-tables
#| tbl-cap: "两个表格并置示例2"
#| tbl-subcap:
#|   - "Cars数据"
#|   - "Pressure数据"
#| layout-ncol: 2

library(knitr)
kable(head(cars))
kable(head(pressure))
```

输出为

```{r}
#| label: tbl-tables
#| tbl-cap: "两个表格并置示例2"
#| tbl-subcap:
#|   - "Cars数据"
#|   - "Pressure数据"
#| layout-ncol: 2

library(knitr)
kable(head(cars))
kable(head(pressure))
```

类似地，通过选项`layout-nrow: 2`就可将二个子图堆叠起来.

```r
#| label: tbl-tables-by-row
#| tbl-cap: "两个表格堆叠示例"
#| tbl-subcap:
#|   - "mtcars数据"
#|   - "iris数据"
#| layout-nrow: 2

library(knitr)
kable(head(mtcars))
kable(head(iris))
```
输出为

```{r}
#| label: tbl-tables-by-row
#| tbl-cap: "两个表格堆叠示例"
#| tbl-subcap:
#|   - "mtcars数据"
#|   - "iris数据"
#| layout-nrow: 2

library(knitr)
kable(head(mtcars))
kable(head(iris))
```
